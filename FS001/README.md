Personnel Profile Page

This project is a simple Personnel Profile Page created using HTML and CSS. It displays the details of an employee such as name, ID, guild, XP earned, completed tasks, join date, performance, and last synced time.

Files

profile.html – Contains the structure of the webpage.
style.css – Contains the styling for the webpage.

How to Run

Open the project folder.

Double-click profile.html or open it using any web browser.
The profile page will be displayed.

Technologies Used

HTML5
CSS3
