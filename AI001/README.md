CSV ANALYZER

MISSION OVERVIEW

    this project is about loading a csv file and generating required analysis report using pandas library

SETUP INSTRUCTIONS:

    Depedndencies required:
        python 3 or above
        panadas library (pip install pandas)

    Setup:
        place legacy_internship_records_final.csv and csv_analyzer.py in the same directory
        open terminal in directory
        and run the python file
        python csv_analyzer.py

OUPUT:

    analysis_report.txt is generated automatically in project directory

    this file contains:
        shape of the data
        data completeness
        entity identifier analysis
        properties names
        missing value analysis
        statistical analysis
        distribution of data
